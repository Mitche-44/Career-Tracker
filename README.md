

# Career Tracker

Career Tracker is a frontend job application tracking system built with React. It helps job seekers organize, track, and manage their job applications through a centralized dashboard, making the job search process more structured and efficient.

This project was built to demonstrate real-world frontend engineering skills, including application state management, routing, authentication flows, and scalable project architecture.

---

## Repository

GitHub: [https://github.com/Mitche-44/Career-Tracker](https://github.com/Mitche-44/Career-Tracker)

---

## Project Overview

Career Tracker allows users to:

* Create and manage job applications
* Track application progress across different stages
* Search, filter, and sort applications
* Maintain notes and details for each application

The application follows modern frontend best practices and mimics how production-level React applications are structured.

---

## Tech Stack

* React (Vite)
* JavaScript (ES6+)
* React Router DOM
* Context API for state management
* CSS or Tailwind CSS
* LocalStorage for session persistence
* Mock data layer (ready for backend integration)

---

## Features

### Authentication (Frontend Simulation)

* User signup and login
* Protected routes
* Session persistence using localStorage
* Logout functionality

### Job Application Management

* Add new job applications
* Edit application details
* Delete applications
* Track application status (Applied, Interview, Offer, Rejected)
* Add notes per application

### Dashboard

* View all applications in one place
* Filter applications by status
* Search by company name or role
* Sort applications by date

### User Experience

* Responsive layout
* Form validation and error handling
* Loading and empty states
* Clean and intuitive UI

## Getting Started

### Prerequisites

* Node.js (v18 or higher recommended)
* npm

### Installation

```bash
git clone https://github.com/Mitche-44/Career-Tracker.git

cd Career-Tracker
npm install
npm run dev
```
